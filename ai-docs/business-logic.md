# Бизнес-логика Smart Response

## Общая концепция

Smart Response — это платформа для создания интеллектуальных форм с автоматической генерацией персонализированного контента на основе искусственного интеллекта. В отличие от традиционных форм-сборщиков, которые только собирают данные, Smart Response анализирует введённую информацию и мгновенно генерирует релевантные ответы в виде текста или изображений.

## Целевая аудитория

### Маркетологи
Создают формы для рекламных кампаний, которые повышают качество лидов за счёт мгновенной генерации персонализированных предложений на основе данных респондента. Формы могут анализировать сайт потенциального клиента и предлагать конкретные решения.

### Продакт-менеджеры
Используют умные опросы для сегментации пользователей и понимания их намерений. Система автоматически анализирует ответы и генерирует инсайты, что ускоряет процесс принятия решений.

### Основатели стартапов
Добавляют функциональные инструменты типа "Получить расчёт" или онбординговые формы без привлечения разработчиков. Платформа позволяет создавать сложные интерактивные формы с AI-логикой через визуальный редактор.

## Решаемые проблемы

### Проблема низкой конверсии форм
Традиционные формы часто имеют низкую конверсию, потому что респонденты не получают немедленной ценности. Smart Response решает это, предоставляя мгновенный персонализированный результат на основе введённых данных.

### Необходимость ручной обработки данных
После заполнения формы требуется ручная обработка данных для генерации ответов или предложений. Платформа автоматизирует этот процесс через AI, экономя время и ресурсы.

### Сложность создания персонализированных взаимодействий
Создание персонализированных форм с динамической логикой требует программирования. Smart Response предоставляет no-code решение с визуальным редактором и AI-логикой.

### Недостаток контекста для генерации ответов
Для качественной генерации контента нужен контекст. Платформа решает это через базу знаний (загрузка файлов), анализ URL сайтов и сбор структурированных данных через кастомные поля.

## Ключевые бизнес-возможности

### Интеллектуальный анализ данных
Система анализирует URL сайтов респондентов, извлекает контент и использует его для генерации релевантных ответов. При блокировке доступа используется прокси-сервис для обхода ограничений.

### База знаний для контекста
Владельцы форм могут загружать файлы (PDF, DOCX, изображения и др.) в базу знаний. AI использует этот контент для более точной и релевантной генерации ответов, учитывая специфику бизнеса или продукта.

### Многовариантная генерация контента
Платформа поддерживает три формата результата:
- Текстовые ответы с форматированием Markdown
- Генерация изображений на основе контекста
- Комбинированный формат (изображение + текст)

### Управление лидами
Все заполненные формы автоматически сохраняются как лиды с полной информацией о респонденте, сгенерированном контенте и статусе обработки. Владельцы форм могут управлять лидами, добавлять заметки и отслеживать прогресс.

### Гибкая настройка форм
Визуальный редактор позволяет создавать формы с различными типами полей: текст, URL, выбор, множественный выбор, чекбоксы, загрузка изображений, заголовки и дисклеймеры. Каждое поле можно настроить как обязательное или опциональное.

## Бизнес-модель и ценность

### Ценность для владельцев форм
- Повышение конверсии за счёт мгновенной генерации ценности
- Автоматизация обработки данных и генерации ответов
- Экономия времени на ручную работу с лидами
- Возможность масштабирования без увеличения операционных затрат

### Ценность для респондентов
- Мгновенное получение персонализированного результата
- Понимание того, что их данные обработаны и проанализированы
- Получение конкретных рекомендаций или предложений вместо простого подтверждения отправки

### Система квот и ограничений
Платформа использует гибкую систему квот для контроля использования ресурсов:
- Лимиты на количество форм (по умолчанию 2 для обычных пользователей)
- Лимиты на количество лидов (по умолчанию 20 суммарно)
- Ограничения хранилища для базы знаний (по умолчанию 50 МБ)
- Дневные лимиты на тестирование форм (по умолчанию 50 раз)

Администраторы могут настраивать квоты для каждого пользователя индивидуально, что позволяет создавать различные тарифные планы.

#### Ограничения на токены AI
Платформа применяет различные ограничения на входные и выходные токены в зависимости от типа генерации:

**Входные токены (input):**
- Текстовая генерация: безлимит — весь контекст отправляется в AI без обрезки
- Генерация изображений: контекст ограничен до 12 000 символов перед созданием промпта
- Извлечение текста из файлов: максимум 50 000 символов на файл (при превышении текст обрезается)
- Контент из URL: безлимит на извлечение текста со страницы

**Выходные токены (output):**
- Текстовая генерация: ограничения определяются моделью OpenAI (явных ограничений в коде нет)
- Генерация промпта для изображения: максимум 1 000 токенов
- Финальный промпт для DALL-E: максимум 3 500 символов (ограничение API DALL-E)

## Уникальные особенности

### Адаптивная генерация на основе контекста
Система комбинирует несколько источников информации для генерации ответа:
- Глобальные промпты (настраиваются администратором)
- Индивидуальные промпты формы
- Контент из URL респондента
- Данные из кастомных полей формы
- Файлы из базы знаний

### Мультимодальная обработка
Платформа поддерживает работу с изображениями: анализ изображений из базы знаний, загрузка изображений респондентами, генерация новых изображений на основе контекста и редактирование существующих изображений.

### Email-интеграция
Автоматическая отправка результатов респондентам и уведомлений владельцам форм о новых лидах. Это обеспечивает непрерывность взаимодействия и не требует ручного вмешательства.

### Публичный доступ к формам
Формы могут быть встроены на внешние сайты через публичные ссылки. Система поддерживает CORS для кросс-доменных запросов, что позволяет использовать формы на любых сайтах.

## Заключение

Smart Response решает фундаментальную проблему традиционных форм — отсутствие немедленной ценности для респондента. Через автоматическую AI-генерацию персонализированного контента платформа превращает формы из инструмента сбора данных в инструмент создания ценности, что повышает конверсию и улучшает пользовательский опыт.
